# 命令行工具

### Windows的CMD命令

   ```
   进入某盘： D:
   查看目录： dir
   查看更多操作：dir /?
   创建目录： md folder_name
   删除目录： rd folder_name
   查看本机ip： ipconfig
   清除： cls
   复制文件： copy 路径\文件名 路径\文件名
   移动文件： move 路径\文件名 路径\文件名
   删除文件： del file_name
   测试网络： ping ip
   查看命令： help
   ```
